Hello :D

This tutorial show how to structure an AngularJS app.

There many ways that you can do this but I will show the way that I found after some researchs.

### 1 Setup dependencies

First we need to setup some dependencies tools to start the development.

If you already has installed nodejs, npm and bower jump to next section.

The first dependency is to install the nodejs (and npm if this is not present) check in the 
[node.js site](http://nodejs.org/) to get info about install.

After npm is installed, we need to install the [bower](http://bower.io/) to
manage our font-end packages.

```
npm install -g bower
```

To test if bower is working run the command `bower` in your terminal to see the
commands list.
